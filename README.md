Client Library for the REST-API of Tesserae
===========================================

This repository provides a thin library wrapper around the REST-API
of [Tesserae](https://github.com/dmellok/tesserae).


Installation
------------

Just copy `tesserae_api.py` to your device. No other libraries are necessary.


Usage
-----

An UI-based application that uses this library as a submodule is
available in my repository
<https://github.com/bablokb/tesserae-device-circuitpython>.
